# 🕌 HIKMAH HUB — Исламская Энциклопедия

## 📘 О проекте
**HIKMAH HUB** — исламская энциклопедия, созданная на базе **Groq Llama 3.1**, объединяющая знания из Корана, Сунны и трудов учёных в современном технологичном формате.  
Проект помогает мусульманам изучать ислам системно, красиво и понятно.

---

## ⚙️ Основные возможности
- 📚 **Энциклопедические статьи** — каждая тема оформлена как страница книги: аят, хадис, толкование, история, практика.  
- 🕌 **50 исламских тем** — Таухид, Иман, Фикх, Ахляк, Судный день и другие.  
- 🔍 **Подкатегории** — подробные разделы внутри каждой темы.  
- 💬 **Умный поиск** — бот отвечает как энциклопедия.  
- 🧠 **Groq AI** — быстрые и точные ответы с исламским фильтром.  
- 🎨 **Tech‑логотип** — современный дизайн с исламским смыслом.
